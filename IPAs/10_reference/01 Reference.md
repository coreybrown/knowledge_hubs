# Reference

The **Reference** domain is the lookup layer of this knowledge base. Where the other
nine domains explain ideas in narrative form, the reference notes are built for fast
retrieval: glossaries, indexes, tables, timelines, and curated resources. Use this
domain when you know *what* you want but not *where* it lives.

> [!note]
> If you are new here, start in [[What Is an IPA]] and [[Home]]. If you are looking
> for a specific note, jump straight to the [[Master Index]].

## The Reference Notes

| [[Master Index]] | A complete map of every note in the vault, organized by all ten domains. The single best starting point for navigation. |
| [[Style Index]] | A focused index of every IPA style, each with a one-line descriptor. |
| [[Timeline of IPA History]] | A chronological table of key milestones from 1700s pale ale to the modern era. |

| [[Recommended Reading and Resources]] | Curated books, websites, and podcasts for going deeper. |
| [[Frequently Asked Questions]] | Quick answers to the most common IPA questions, each linking onward. |

## How to Use This Domain

> [!tip]
> - **Don't know a word?** → [[Glossary of IPA Terms]]
> - **Looking for a note?** → [[Master Index]]
> - **Comparing styles?** → [[Style Index]] and [[IPA Style Comparison Table]]
> - **Want dates and numbers?** → [[Timeline of IPA History]] and [[IPA Beer Statistics and Data]]
> - **Ready to read a real book?** → [[Recommended Reading and Resources]]
> - **Just have a question?** → [[Frequently Asked Questions]]

The reference notes are deliberately link-dense. Almost every term and figure points
back into a narrative note that explains it in full. Treat this domain as the index
at the back of a book — concise here, expansive elsewhere.
