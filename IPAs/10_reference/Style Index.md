# Style Index

Every IPA style in this knowledge base, each with a one-line descriptor. This index
covers the `03_styles` domain. For styles set side by side on shared metrics, see the
[[IPA Style Comparison Table]]; for the full vault, see the [[Master Index]].

> [!note]
> The IPA is a *family*, not a single recipe. For how these styles descend from one
> another, see the [[IPA Family Tree]].

## Core Styles

| Style | One-line descriptor |
|---|---|
| [[English IPA]] | The malt-balanced original — earthy, floral English hops over biscuity malt. |
| [[American IPA]] | The modern benchmark — citrus and pine hops over a clean, dry base. |
| [[West Coast IPA]] | Crystal-clear, assertively bitter, and bone-dry. |
| [[New England IPA]] | Soft, hazy, and juicy with low bitterness and big aroma. |
| [[Double IPA]] | High strength and high hop intensity — typically 7.5–10% ABV. |
| [[Triple IPA]] | The extreme end — 10%+ ABV with relentless hop saturation. |
| [[Session IPA]] | Full hop character at a low, sub-5% ABV. |

## Variant and Specialty Styles

| Style | One-line descriptor |
|---|---|
| [[Black IPA]] | Dark, roast-tinged malt paired with bold hop bitterness. |
| [[White IPA]] | An IPA crossed with witbier — wheat, spice, and citrus hops. |
| [[Belgian IPA]] | Expressive Belgian yeast esters layered onto a hoppy base. |
| [[Rye IPA]] | Spicy, peppery rye sharpening the malt backbone. |
| [[Red IPA]] | Caramel-rich amber malt balanced against American hops. |
| [[Brut IPA]] | Bone-dry and effervescent, made champagne-crisp with enzymes. |
| [[Milkshake IPA]] | Lactose, fruit, and vanilla for a creamy, dessert-like beer. |
| [[Cold IPA]] | Brewed with lager yeast and adjuncts for a crisp, clean finish. |
| [[Brett IPA]] | Wild *Brettanomyces* yeast adding funky, fruity complexity. |
| [[Sour IPA]] | Kettle-soured tartness combined with juicy hop aroma. |
| [[India Pale Lager]] | The lagered cousin — hop-forward but cold-fermented and crisp. |
| [[Specialty and Experimental IPAs]] | The fringe — pastry, brut-meets-hazy, and one-off experiments. |
